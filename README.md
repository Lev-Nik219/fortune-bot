# 🤖 Fortune Bot - Telegram Бот-Предсказатель

Волшебный Telegram бот, который создает персональные предсказания и гороскопы с помощью искусственного интеллекта.

## ✨ Возможности

- 🎯 Персональные предсказания с учетом имени, пола и вопроса
- 🌟 Уникальные гороскопы для всех знаков зодиака
- 💰 Оплата через CryptoPay (0.10 USDT)
- 🤖 Генерация через Hugging Face AI
- 📊 Статистика и история предсказаний
- 🔄 Автоматическая проверка оплаты

## 🚀 Технологии

- Python 3.11
- python-telegram-bot
- Flask
- Hugging Face Inference API
- CryptoPay API
- SQLite
